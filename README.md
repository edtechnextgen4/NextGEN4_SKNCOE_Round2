Peerify – Peer-to-Peer Skill Validation Platform
https://peerify.netlify.app/

Peerify is a clean, intuitive multi-page web platform that demonstrates how students and professionals can validate their skills using a community-driven review ecosystem. 
The prototype showcases the entire user flow — login, skill submission, peer reviews, dashboards, leaderboards, profiles, and more — all crafted using lightweight frontend technology and deployed on Netlify.

------------------------------

FEATURES

Home & User Flow
- Modern landing page introducing the platform concept.
- Smooth navigation to all major modules.

Login Interface
- User-friendly login screen.

Skill Submission
- Interface to showcase skills and upload details.

Peer Review Panel
- Console for reviewing submissions with structured design.

Dashboard
- Shows activity, progress, and quick navigation options.

Profile System
- Displays badges, skills, and user information.

Leaderboard
- Ranking interface to encourage engagement.

UI & Deployment
- Multi-page modular design.
- Clean CSS styling.
- JavaScript-driven interactions.
